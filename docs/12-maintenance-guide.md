# 12 - Non-Technical Store Owner Maintenance Guide

## How to Manage Your Store

### 1. How to Add a New Product
1. Log in at `http://localhost:3000/account` with `admin@dormlights.com` / `admin123`.
2. Click **Admin Panel** in the top navigation bar.
3. Select **Products Catalog** on the left menu.
4. Click the golden **+ Add New Product** button in the top right.
5. Fill in the product title, category, price, stock units, image URL, and click **Create Product**.

### 2. How to Update Order Delivery Status
1. Navigate to **Customer Orders** in the Admin sidebar.
2. Find the student order by ID or name.
3. Click the status button (**Shipped** or **Delivered**) to update the live customer status.

### 3. How to Create Promo Discount Codes
1. Navigate to **Coupons & Promos**.
2. Type your promo code (e.g. `WELCOME20`), set discount percentage, and click **Generate Coupon Code**.
