# 05 - Customer Features & Interactive Guide

## Storefront Functional Features
1. **Live Search**: Instant keyword search modal with auto-redirection to filtered catalog view.
2. **Catalog Filters**: Filter products by category, max price slider, star rating, and spec tags.
3. **Product Detail Variants**: Select length (1.5m, 3m, 5m) and color temperature (2700K Warm Gold, 4000K Neutral, RGBIC Neon Zone).
4. **Interactive Room Simulator**: Real-time lighting preview widget allowing students to simulate brightness and color temperature output on dorm backdrops.
5. **Cart & Sliding Drawer**: Slide-over cart drawer with free shipping progress bar, promo code validation, quantity increment/decrement, and subtotal calculation.
6. **Checkout Flow**: Complete shipping form, mock payment gateway (Credit Card & Apple Pay), and instant order receipt confirmation with order ID (`ORD-XXXX`).
7. **Wishlist & Account Portal**: Heart toggle to save favorite products to local storage and view past order history.
8. **WhatsApp Support Integration**: Floating click-to-chat widget for direct communication with the DormLights student support desk.
